Sniper
======
##实用、功能强大、高性能的HTTP负载工具
##采用Golang编写

##功能
- GET / POST
- keep-alive模式
- https
- 图表展示结果
- 测试多个目标
- 支持大文件负载

##性能
- 内存占用略高于Apache Benchmark（ab）
- 执行速度接近ab，高并发时超过ab
- 支持10k以上并发

##图表展示
- 统计分析每个请求
- 输出建立连接时间
- 输出服务端响应时间
- 输出总时间
